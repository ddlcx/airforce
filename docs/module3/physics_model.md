## 2. 羽毛球空气动力学 (Cohen et al. 2015 [1])

### 2.1 阻力模型

羽毛球被视为受重力和空气阻力作用的质点。其重心运动遵循向量形式的二阶 ODE [1, §2.1]：

```
M d𝐔/dt = M𝐠 − (1/2) ρ S C_D |𝐔| 𝐔                                  (1)

其中:
  𝐔 — 速度向量 (m/s), |𝐔| 为其模值
  𝐠 — 重力加速度向量 (指向地面)
  M — 羽毛球质量 (kg)
  ρ — 空气密度 (kg/m³)
  S — 迎风截面积 π(D/2)² (m²)
  C_D — 阻力系数 (无量纲)
```

Cohen (2015) 的风洞实验确认以下关键事实：
- C_D 在 Reynolds 数 Re = 10⁴ ~ 2×10⁵ 范围内近似常数 [1, Fig.12]
- 羽毛球（feathered）C_D = 0.65 ± 0.05，塑料球 C_D = 0.68 ± 0.05 [1, §2.2]
- 旋转对 C_D 无显著影响（C_D 保持在 0.65~0.75 之间），但有助于限制进动、提高飞行稳定性 [1, Fig.16]
- 风洞实验确认无升力：当羽毛球轴线与速度方向对齐时升力为零 [1, §1]

**翻转动力学的排除**: 羽毛球击打后会经历"翻转"（cork 从后方旋转到前方），Cohen [1, §3.1] 推导了翻转角度的阻尼振荡 ODE。翻转的稳定时间 τ_s ≈ 100 ms 远短于飞行时间 τ₀ ≈ 1 s，翻转完成后球轴线与速度方向对齐，此后纯阻力质点模型适用。本模块的 ODE 模型假设球已完成翻转，处于稳定飞行状态 [1, §3.1, Fig.18]。

**环境因素**: 温度通过空气密度 ρ 影响气动长度 𝓛，10°C 到 40°C 间飞行距离可变化约 10% [1, §3.3]。湿度主要影响羽毛球（鹅毛吸水增重），对塑料球影响较小。本模块采用标准条件 ρ = 1.2 kg/m³ (20°C)，实际使用中可通过调节 c_d 隐式吸收环境差异。

### 2.2 集总阻力参数与特征量

将式 (1) 除以 M，得到 MonoTrack [2] 所用形式：

```
d𝐔/dt = 𝐠 − c_d |𝐔| 𝐔                                                (2)
```

特征长度（气动长度）𝓛 和集总阻力参数 c_d 的定义：

```
𝓛  = 2M / (ρ S C_D)      ← 气动长度 (m)
c_d = 1 / 𝓛 = ρ S C_D / (2M)   ← 集总阻力参数 (1/m)
```

| 物理量 | 数值 | 来源 |
|--------|------|------|
| c_d 标称值 | 0.218 (1/m) | Cohen [1]: C_D=0.65, M=5g, S=28cm² |
| 气动长度 𝓛 | 4.6 m | Cohen [1, §2.1] |
| 终端速度 U_∞ = √(g𝓛) | 6.7 m/s | Cohen [1, §2.1] |
| 最大记录速度 U_max | 137 m/s | Cohen [1, §3.2] |

**射程饱和特性**: 对于高远球，Cohen [1, §2.3] 给出射程的解析近似 x₀ = (𝓛/2) cosθ₀ · ln(1 + 4U₀²sinθ₀/(g𝓛))。当 U₀ 很大时射程呈对数增长趋于饱和，这解释了为何羽毛球即使被大力击打也难以飞出 13.4 m 长的球场。

c_d 作为 MHE 的待估参数，允许在不同飞行段之间变化。变化来源包括：羽毛球在回合中逐渐损耗导致截面积和 C_D 改变 [1, §2.2, Fig.13]，以及环境温湿度差异。

**塑料球与羽毛球差异**: 塑料球质量约 5.3 g（vs 羽毛球 5.0 g），C_D ≈ 0.68 [1, §2.2]，导致 𝓛 更大、飞行距离更远、轨迹不如羽毛球"尖锐"。c_d 作为待估参数可自动适配两种球类型。

## 3. 飞行平面简化

### 3.1 原理

无升力条件下，羽毛球轨迹处于包含初速度方向和重力方向的垂直平面内。定义该平面的方位角 ψ，3D 运动简化为 2D 飞行平面 (s, z) 中的运动：

```
        z (高度)                       俯视图:  Y_w
        ▲                                      ▲
        │     ·  ·                              │    /
        │   ·      ·   轨迹                     │   /  ψ
        │ ·           ·                         │  /
        │·               ·                      │ /
        └───────────────────→ s              (x0w,y0w) ───→ X_w
                                               起点     s·cos(ψ)

飞行平面 → 3D 世界坐标映射:
    X_w = x0w + s · cos(ψ)
    Y_w = y0w + s · sin(ψ)
    Z_w = z
```

### 3.2 飞行平面 ODE

```
物理状态: x_phys = [s, z, v_s, v_z]    (4维)

ds/dt   = v_s
dz/dt   = v_z
dv_s/dt = −c_d · √(v_s² + v_z²) · v_s                                (3)
dv_z/dt = −g − c_d · √(v_s² + v_z²) · v_z                            (4)
```
